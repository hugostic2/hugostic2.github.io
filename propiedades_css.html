Generic Font Family

En este caso escribimos en la etiqueta style font-family:sans-serif por ejemplo y lo aplicaré a todo body

Specific Font Family

Escribiremos la familia especifica monospace para este párrafo y en el caso de que no sea posible como segunda opcion serif

Font Size

Escribiremos todos los encabezados h2 en el mismo tamaño

Font-style

Esta línea estará en cursiva

Line-Height

Esta línea tendrá la misma altura de línea

Text-Align

Alinearé los h2 para que estén centrados

Text-Decoration

En este caso subrayaremos todos los h2
